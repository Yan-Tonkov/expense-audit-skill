Выполни аудит расходов Google Sheets по шагам:

## 1. Получи Sheet ID

Аргумент: `$ARGUMENTS`

Извлеки Sheet ID:
- URL вида `https://docs.google.com/spreadsheets/d/<ID>/edit` → возьми `<ID>`
- Если передана строка без слешей — используй как есть

## 2. Запусти аудит

```bash
node scripts/expense-audit.js <SHEET_ID>
```

Скрипт сам:
- Загружает транзакции через gws CLI
- Считает итоги по месяцам и категориям
- Выявляет выплаты уволенным сотрудникам (ghost payroll)
- Записывает отчёт в лист «Аудит расходов» того же документа

## 3. Сообщи пользователю

После выполнения выведи:
- Краткое summary: общая сумма, период, топ-категории
- Ключевые аномалии (ghost payroll, дубли, выбросы)
- Ссылку на таблицу с готовым отчётом
